🚀 Erste Schritte
Anmelden, erstes Board, erste Karte
Anmelden
Du erhältst von einer einladenden Person einen Zugangs-Link und ein Passwort per Mail (zwei getrennte Mails aus Sicherheitsgründen). Öffne den Link, melde dich mit E-Mail-Adresse und Passwort an.
Erstes Board anlegen
Oben in der Tab-Leiste auf + klicken, einen Namen vergeben. Ein neues Board startet mit einer Standard-Spalte — weitere Spalten und Lanes lassen sich jederzeit über die +-Buttons an den passenden Stellen ergänzen.
Erste Karte
In einer Spalten-Zelle auf + Karte klicken, Titel eingeben, Enter — fertig. Ein Klick auf die Karte öffnet das Detail-Modal mit allen weiteren Optionen (Notiz, Fälligkeitsdatum, Anhänge, Labels, Zuweisung).
Board-Optik anpassen
Über das Optik-Symbol in der Kopfzeile lässt sich Hintergrundfarbe oder -bild pro Board einstellen — rein zur Wiedererkennung, keine Funktion.
Alles wird automatisch gespeichert — es gibt keinen „Speichern"-Knopf. Änderungen sind sofort für alle Mitglieder sichtbar, die das Board gerade geöffnet haben.
