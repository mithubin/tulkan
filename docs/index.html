tulkan

Organisation, Vorbereitung und Dokumentation für Teams, Klassen und Projekte — vom einfachen Kanban-Board bis zur vollwertigen Arbeitsumgebung mit eigener Datensicherung.

Selbstgehostet, quelloffen, ohne Cloud-Abhängigkeit. Kein Vendor-Lock-in, keine Abo-Falle — Ihre Daten bleiben auf Ihrem eigenen Server.

Ein Werkzeug, das mit Ihnen mitwächst

Man kann tulkan wie ein einfaches Trello-artiges Board benutzen — und genau dort aufhören. Wer mehr braucht, findet darunter eine komplette Arbeitsumgebung, die mitwächst statt im Weg zu stehen.

📋 Einfaches Kanban
Boards, Spalten, Karten, Drag & Drop — sofort nutzbar, keine Einarbeitung nötig. Genau das, was jeder von einem Kanban-Board erwartet.
🗂️ Tiefere Struktur, wenn sie gebraucht wird
Lanes quer über die Spalten, Unterkarten bis zu drei Ebenen tief, Dateikarten als vollwertige Karten — die Struktur wächst mit der Komplexität der Arbeit, statt sie von Anfang an aufzuzwingen.
🧩 Meta-Werkzeuge direkt an Karte und Datei
Jede Karte und jede Datei kann eigene Notizen (Markdown), Unteraufgaben und eine eigene Tabelle tragen — wahlweise als freie Tabelle oder als Drei-Zustands-Pipeline. Kontext und Inhalt bleiben an einem Ort statt über drei verschiedene Werkzeuge verstreut.
🕐 Personen- & Zeit-Dokumentation — das „Klassenbuch"
Anwesenheit, Punkte und Zeitverläufe lassen sich über einen Personen-Bezug pro Karte festhalten. Snapshots frieren den Zustand eines Boards zu einem bestimmten Zeitpunkt ein — eine echte, retrospektiv abrufbare Verlaufsdokumentation, kein loser Haufen Notizen.
🖼️ Präsentationsfähig direkt aus der Karte
Mehrere PDFs, Bilder und Videos an einer einzigen Karte — direkt im Browser als Galerie durchblätterbar, im Vollbild, ohne Download. Eine Karte wird so zur vollwertigen Präsentationsgrundlage, nicht nur zur Ablage.
✏️ Grundbearbeitung, ohne die Seite zu verlassen
Office-Dokumente (docx/xlsx/pptx/odt) direkt im Browser bearbeiten, Bilder zuschneiden und drehen, Text- und Markdown-Anhänge inline editieren, Skizzen direkt einbetten.
✉️ E-Mail — einzeln und als Serie
Eine Mail direkt aus einer Karte heraus versenden — oder das ganze Board als Datenquelle für eine Serienmail nutzen: Platzhalter aus einer Tabelle befüllt, an alle passenden Empfänger auf einmal, mit Versandprotokoll.
📄 Seriendokumente
Eine DOCX-Vorlage plus Datenquelle genügt: pro Datensatz entsteht automatisch ein fertiges Dokument — Serienbriefe, Zertifikate, Berichte, ohne manuelles Copy-Paste. Serienmail und Seriendokument lassen sich in einem einzigen kombinierten Arbeitsschritt zusammen ausführen.
🔧 Und dann: die tul_s-Werkzeuge
Eine Datei, die schon in einer Karte liegt, muss nicht heruntergeladen werden, um sie weiterzuverarbeiten. Direkt verbunden: Audio-Transkription, PDF-Optimierung, Lernkarten-Erzeugung, Brennofen-Kurvenauswertung, Bildlayout und mehr — acht eigenständige Werkzeuge, eine gemeinsame Anmeldung, eine Datei-Brücke ohne Umweg. Nicht jeder braucht eine Ofenkurven-Auswertung — aber das ist genau der Punkt: tul_s ist keine feste Liste, sondern ein Baukasten. Mit etwas Programmierkenntnis — oder heute realistisch: mit KI-Unterstützung — lässt sich die Toolbox nahezu unbegrenzt um eigene, zugeschnittene Mini-Apps für genau die Projekt- und Dateiaufgabe erweitern, die gerade ansteht. Ergebnisse fließen per Datei-Brücke direkt zurück auf die auslösende mkan-Karte.

Struktur- und Datensicherung inklusive

Nichts davon nützt etwas, wenn es nicht sicher ist. Deshalb ist das kein Zusatz, sondern Grundausstattung.

🔐
Vier Rollen, fein abgestuft
Viewer · Editor · Owner · Admin — jede Ebene bekommt genau die Rechte, die sie braucht.
🚪
Externer Zugang auf eine einzige Spalte
Eine Person von außen sieht und bearbeitet ausschließlich ihre zugewiesene Spalte — kein Einblick ins restliche Board, keine Verschiebungen, keine Board-weiten Aktionen.
💾
Automatische Backups
Datenbank-Sicherung vor jedem Update, Snapshots als zusätzliche inhaltliche Zeitkapsel — zwei unabhängige Sicherheitsnetze.
🗝️
Zwei-Kanal-Sicherheit bei Einladungen
Zugangslink und Passwort-Erinnerung gehen als getrennte Mails raus — eine abgefangene Nachricht reicht allein nicht für einen Zugriff.
📦
Board als durchsuchbares Archiv exportieren
Ein Klick erzeugt ein ZIP mit der kompletten Board-Struktur als Ordnerbaum — Notizen, Titelbilder, Anhänge, Tabellen, alles mit lesbaren Dateinamen. Lesbar auch ganz ohne laufendes mkan, mit jedem Zip-Programm. Ergänzt (nicht ersetzt) die automatischen Datenbank-Backups als inhaltliche, jederzeit einsehbare Zeitkapsel.
Alles davon aus zwei Browser-Tabs

Kein App-Store, keine Installation auf jedem Gerät, kein Update-Zwang für Nutzende. Ein Tab für das Board, ein Tab für die Werkzeuge — der Rest läuft auf dem eigenen Server, den sonst niemand sieht.
