IDs & Verknüpfungen
SQLite pflegt intern eine unsichtbare rowid — die braucht niemand selbst anzulegen.
Für Verknüpfungen zwischen Tabellen reicht eine gemeinsame Spalte:
z.B. "Kürzel", "Name" oder "SID" in beiden Tabellen.
Soll eine laufende Nummer sichtbar sein, einfach beim CSV-Export als erste Spalte mitgeben.
-- Verknüpfung über Kürzel:
SELECT s."Name", n."Note"
FROM "Schüler" s
JOIN "Noten" n ON s."Kürzel" = n."Kürzel"
ORDER BY s."Name" COLLATE NOCASE
