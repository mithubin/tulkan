⚙ Technik & Daten
Multi-Kanban ist eine einzelne HTML-Datei — kein Server, kein Framework, kein CDN, keine externe Abhängigkeit. Alles läuft vollständig im Browser.
DateiEine einzige .html-Datei, selbst-enthaltend. Kein Build-Schritt, keine Installation.
SpeicherPrimär via File System Access API (Chrome/Edge): schreibt direkt in eine .json-Datei auf deiner Festplatte. Firefox-Fallback: localStorage.
DatenBoards, Karten, Swimlanes, Anhänge und Konfiguration liegen ausschließlich lokal — in der JSON-Datei oder im localStorage.
NetzwerkKeine einzige Netzwerk-Anfrage. Kein Tracking, keine Telemetrie, keine externen Ressourcen.
OptikTheme-Einstellungen werden im localStorage gespeichert — gerätebezogen, nicht in der JSON.
BrowserVoll funktionsfähig in Chrome/Edge. Firefox: kein direkter Dateizugriff — nur Import/Export. Anhänge und Titelbilder füllen den localStorage (5–10 MB) schnell; bei intensiver Nutzung ist der Export-Fallback unzuverlässig.
